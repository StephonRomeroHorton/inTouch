# InTouch 🌎

A minimal social platform where users share how they feel and why — without likes, comments, or engagement metrics.

Built as a full-stack JavaScript application using React, Node.js, Express, and MongoDB with a simple REST API architecture.

🔗 Live Demo: [(https://feelings-app.netlify.app/)]

---

## About The Project

InTouch is a lightweight semi–social media platform focused on emotional expression rather than engagement.

Users can post:

- how they feel  
- why they feel that way

and view how others around the world are feeling.

Unlike traditional social media platforms, InTouch intentionally removes likes, comments, and follower systems. The goal is to create a space focused purely on expression and shared human experience.

The platform also includes a mental health news feed powered by a third-party API to surface relevant information and resources.

This project was originally an unfinished early learning project that I'm rebuilding live to demonstrate my growth as a developer and properly implement full-stack architecture.

## Architecture

The application follows a simple full-stack REST architecture:

```
React Frontend
      ↓
Express API Server
      ↓
MongoDB Database
```

User submissions are sent through API routes, stored in MongoDB, and displayed in a public feed.

The application also integrates a third-party news API to provide mental health related articles alongside user posts.
